AR Example 1

Description

This is a dummy 3D plant example file that has been sourced for test purposes.

You can see a greater level of detail within this example, such as:

  • Crisp Texture

  • Shadow Depth

  • The ability to rotate a full 360 degrees


To achieve something similar for you product range, each product would need to be created in a 3D model format. This can be done by hiring an external 3D modeling company who can use a range of techniques to create the appropriate files that you will need.

AR Example 2

Description

Above is an example of a current product from your website, to achieve the above you would need to do the following.

Step 1 - Remove any background from the image, leaving the image as a transparent .PNG file type.

Step 2 - Leverage a third party platform such as, https://cartmagician.com to convert your 2D image using one of their templates to a 3D model.

*You will notice there is no depth on this 3D model like example 1, this is because the 3D model is based on a flat 2D image.

Step 3 - Once you have converted the 2D image to the 3D model, you will download the file that is generated and upload this to the website.
